1️⃣ Voice Assistant (assistant.py)
A simple voice-controlled assistant built using Python.  
It can take voice commands and perform basic tasks such as responding, searching, or executing actions.

Features:
- Speech recognition
- Converts voice input to text
- Performs actions based on commands
